# face_detecrt_app

### マークザッカーバーグ、ビルゲイツ、スティーブ・ジョブズの３人を識別するアプリケーションを作成

### 概要
 kerasを使って畳み込みニューラルネットワークを構築し、約各人200枚の画像をモデルに学習。<br>
 顔の検出にはOpenCVを使用した。
